How to Navigate¶

Use the sidebar navigation on the left to browse through different documentation sections:

  • Click section headers (Playground, Routines, etc.) to expand and view available pages
  • Active sections remain expanded when you're viewing pages within them
  • Current page is highlighted with a colored indicator for easy reference
  • Browser search functionality - Use Ctrl/Cmd+F to find specific terms across the current page

Feedback & Contribution¶

The UAW documentation is continuously evolving. If you find areas that need clarification, spot errors, or have suggestions for improvement:

  • Documentation issues can be reported through the main UAW repository
  • Each page is generated from source materials that can be updated by contributors
  • The system supports multiple output formats and automatic regeneration
