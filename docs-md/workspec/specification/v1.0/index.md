# WorkSpec v1.0 (Legacy Reference)

WorkSpec v1.0 is the legacy, pre-versioned simulation schema previously used by UAW simulations.

WorkSpec v2.0 introduces a required `simulation.schema_version` and a breaking structural change (`world`/`process` split). v1.0 documents should be migrated to v2.0:

- Migration guide: [/docs/workspec/guides/migration](/docs/workspec/guides/migration)
- v2.0 overview: [/docs/workspec/specification/v2.0/](/docs/workspec/specification/v2.0/)

This section exists to preserve legacy documentation and to help with migration audits.
